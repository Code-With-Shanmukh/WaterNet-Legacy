# WaterNet-Legacy
Legacy Django-based water quality monitoring and prediction system for drinking and irrigation using machine learning. Preserved as the baseline for future modernization.

WaterNet-Legacy is the original Django-based implementation of a web application for monitoring and assessing water quality for drinking and irrigation purposes using machine learning techniques.

The application provides dataset management, model training and testing, water quality prediction, user management, and reporting features through separate interfaces for administrators, service providers, and remote users.

This repository preserves the original implementation while serving as the foundation for a complete modernization using the latest Python ecosystem.

## Features

- User Authentication
- Administrator Dashboard
- Service Provider Dashboard
- Remote User Dashboard
- Water Quality Dataset Management
- Machine Learning Model Training
- Model Testing
- Water Quality Prediction
- Prediction Reports
- Accuracy Analysis
- Detection Ratio Analysis
- Dataset Download
- User Management
